PySky

<b>Usage</b>

    ./pysky.py [celestial body (string)] [pixel width (int)] [pixel height (int)] [resolution (float)] [brightness scale]


<b>Brightness Scale</b>

    Linear
    Sqrt
    HistEq

<b>Examples</b>

    ./pysky.py andromeda 1280 720 3.5 Linear
    ./pysky.py vega 1920 1080 3.0 HistEq
